# Usine Mouna - Backend

## Prérequis
- Node.js
- MongoDB installé et lancé en local

## Installation

1. Placez-vous dans ce dossier :
   ```bash
   cd backend
   ```
2. Installez les dépendances :
   ```bash
   npm install
   ```
3. Lancez le serveur :
   ```bash
   node app.js
   ```

L'API sera accessible sur http://localhost:5000
